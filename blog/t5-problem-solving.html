Problem Solving

What was the problem, and how did I solve it?

On Saturday, I got stuck on JavaScript while creating a loop. A loop is a mathematical equation which allows you to input variables, meaning you can make slight changes to the outcome. One way a loop can be used is for counting the number of clicks on different variables, which is what this particular task involved. I was blocked on writing the variable inputs for the loop. The solution was a simple change - deleting the statement ‘else’, and just using ‘if’ as a way to identify the variables.
