Whats all the hype about responsive webpages?

With a range of devices viewing webpages it is important to note that each of these devices have a different range of pixels, so webpages must be able to adapt to these different sizes. The hype surrounding Responsive Webpages are about how the webpage adapts it's size, rather than remaining fixed.
